# task_priority_kinematic_control

ROS 2 package that provides a whole-body task-priority kinematic controller for CIRTESUB and dual Alpha manipulators.

## Contents

- Controller plugin for `ros2_control`
- Runtime node for task management
- Custom messages and services
- Example configuration and launch files

## Dependencies

This package depends on ROS 2 control components and project-specific interfaces such as `sura_msgs`.

## Build

Build it inside a ROS 2 workspace with `colcon`.

## Notes

This repository is intended to contain the `task_priority_kinematic_control` package only.
